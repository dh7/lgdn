Le recyclage des déchets est très souvent imparfait. Il doit reposer sur un tri précis et si possible détaillé à la source. Comment motiver les particuliers à le faire. Suggestions  le LOTO DECHET :  •	Distribuer en mairie des sacs de couleurs différentes et NUMEROTÉS avec CODE BARRE, qui seront attribués aux particuliers les demandant. •	Collecter les déchets triés dans les sacs adaptés. •	Vérifier que le tri est correctement effectués et SCANNER LES CODE BARRES des sacs corrects. •	Tirer au sort tous les mois ou toutes les semaines un CODE BARRE parmi les sacs corrects.  Vous verrez que le % de sacs très bien triés sera impressionnant. Le recyclage des déchets devrait pouvoir devenir beaucoup plu large et plus rentable économiquement. C’est la logique de la MOTIVATION POSITIVE.  Tous les déchets recyclés POLLUERONT beaucoup MOINS. Bien des DÉCHETS recyclés permettront de FOURNIR DE L’ÉNERGIE.
