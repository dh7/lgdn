Principe intangible : les pollueurs doivent être les payeurs. Les comportements recherchés (moindre production de CO2, etc …) doivent être induits par les taxes sur les hydrocarbures : sans ce caractère « punitif », pas d'effet rapide sur les comportements. <br><br> Les automobilistes défavorisés parmi les payeurs peuvent être aidés par ailleurs.
