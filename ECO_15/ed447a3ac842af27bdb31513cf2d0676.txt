Dans les zones rurales, il existe souvent un ramassage scolaire avec un mini-bus pour transporter les enfants dans la commune où sont regroupées les classes. Ce mini-bus n'est utilisé que le matin et le soir.  Une fenêtre dans la journée pourrait être utilisée pour transporter les personnes âgées ou les personnes n'ayant pas de véhicule pour faire leurs courses dans la ville voisine (transport  dans une plage horaire définie mais à la demande sur inscription à la mairie) Le véhicule existant ne générerait pas de surcoût  énorme pour la municipalité et pourrait compléter le salaire de l'employé(e). Mieux former les personnels de mairie dans les zones rurales afin qu'une aide puisse être apportée sans que les administrés soient obligés de faire des kilomètres. Améliorer les transmissions internet. Tout faire pour garder les petits commerces dans les villages.
