Je pense que de manière générale, il faudrait augmenter les taxes sur les produits superflus : - tous les produits sucrés, les plats préparés, l'alcool, etc. - les produits non essentiels comme les jeux vidéos, les consoles, les télés, les téléphones haut de gamme, les tablettes, les sèche-linge, etc. et diminuer les taxes sur les produits de première nécessité et les produits biologiques et écologiques : - farine, beurre, œufs, les fruits et légumes, les produits laitiers nature, etc. - lave-linge, etc.
