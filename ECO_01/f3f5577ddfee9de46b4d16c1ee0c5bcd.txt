Taxer la pollution (toutes les pollutions). Les producteurs, via l'achat de "permis de polluer" par exemple (à payer tous les mois en fonction du type de pollution, de l'importance du produit, etc.). Taxer les consommateurs également, via par exemple une taxe sur la pollution sur chaque produit. Taxer les emballages. Instaurer une "taxe kilomètre" qui taxe le nombre de kilomètres parcourus par le produit depuis la création jusqu'à sa distribution. Investir dans le recyclage. Subventionner les initiatives de recyclage, de production sans déchet (zéro déchets), etc. Éduquer les jeunes générations aussi évidemment. Investir dans le télé-travail.
