Pour les professionnels : inciter les entreprises à faire évoluer leurs modes de fonctionnement en tenant compte de l'environnement. Récompenser les entreprises qui font des efforts pour l'environnement (réduire les transports, circuit court, privilégier les énergies propres, le transport FRET,...) et taxer les autres. Inciter les entreprises à privilégier le transport FRET afin de réduire le nombre de camions sur les routes. Encourager les producteurs/fournisseurs divers à réduire les emballages. Pour les particuliers : encourager les particuliers à utiliser les transports en communs, le train, le covoiturage, … afin de réduire le nombre de voitures (réseaux mieux dimensionnés, solutions de transport "porte-à-porte", prix attractifs, …). Réduire les emballages dans la consommation de tous les jours, privilégier les achats en vrac. Des prix plus avantageux pour les produits issus du circuit court, ne pratiquant pas le suremballage, bio, avec un mode de fabrication respectueux de l'environnement. Agriculture : interdire une fois pour toute les pesticides et autres produits ayant des conséquences dramatiques sur l'environnement, même si cela doit réduire la production en permettant aux producteurs respectant l'environnement d'être compétitifs (taxes plus élevées pour les autres producteurs)
