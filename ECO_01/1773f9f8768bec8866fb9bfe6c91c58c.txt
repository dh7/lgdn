Il faut arrêter de vouloir faire supporter les contributions écologiques aux spectateurs plutôt qu'au acteurs. Mais également sensibiliser les spectateurs. La taxe sur les transports aériens est une bonne chose, mais elles devrait être plus dissuasive pour les rencontres sportives internationales. Déplacer les équipes, les coachs, les familles, les dirigeants, les invités, les supporters, les équipes médicales et les sponsors ... là ont est plus dans les valeurs sportives mais dans la pollution gratuite sans bénéfice pour la planète.
