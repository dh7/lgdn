Il faudrait commencer par l'interdiction de tuer le peu d'animaux sauvages qui restent.  Il faudrait aussi interdire le lâcher de certains oiseaux, type perdrix, faisans. Je vois les chasseurs les lâcher les vendredis pour les abattre avec un fusil le lendemain ou le surlendemain. Sans compter les marcheurs qui ne s'aventurent plus à se promener dans les bois les weekends ! Et pour des raisons de sécurité : - obliger, comme pour tous les sportifs, les chasseurs à fournir un certificat médical pour pouvoir se promener avec une arme, - faire effectuer des contrôles d'alcoolémie.  Idem en ce qui concerne les pêcheurs, pour les lâchers de truites qui ont lieu le matin, les truites sont ensuite pêchées l'après-midi-même.
