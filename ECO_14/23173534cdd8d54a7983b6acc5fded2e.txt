Agir dans le cadre des organisations internationales pour soutenir la lutte contre les émissions de CO2. Pour l'aviation, travailler avec les agences internationales pour que le Kérosène soit taxé dans tous les aéroports. Même chose pour le trafic maritime. Prendre le leadership pour que des standards soient rapidement adoptés pour les véhicules électriques et que les recharges soient possibles dans tous les pays avec une même prise.
