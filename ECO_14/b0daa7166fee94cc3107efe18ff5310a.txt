La France doit être un exemple, dans tous les domaines. Les Français, dans leurs comportements, doivent tous être exemplaires, et en premier lieu leur Président, les membres du gouvernement et tous les élus.  Rédiger un code environnemental, au même titre que le code civil ou le code pénal, où seraient condensées toutes les lois régissant la protection et le respect de la Nature. Punir plus sévèrement les pollueurs.
