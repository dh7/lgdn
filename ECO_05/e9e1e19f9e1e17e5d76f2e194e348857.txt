Je ferme le robinet d’eau pendant que je me brosse les dents ; je prends des douches et ferme le robinet pendant que je me savonne ; je suis végétarienne ; je trie les déchets ; je remplis les bouteilles d’eau plutôt que d’en acheter de nouvelles ; je roule tranquillement en voiture ; je mets un pull plutôt qu’augmenter le chauffage ; j’entretiens mes appareils de chauffage ;  j’œuvre pour la protection animale ; j’alerte autour de moi.  Mais ceci est bien insuffisant comparé aux gros pollueurs
